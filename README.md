## Installation
Add this line to your application's Gemfile:

```ruby
gem 'form_creation'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install form_creation
```
## Migration
$ rails form_creation:install:migrations

$ rails db:migrate

## Routes
mount FormCreation::Engine => "/form_creation"
